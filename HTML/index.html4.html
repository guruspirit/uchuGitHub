Типы элементов
  • Блочные: ( div, h1, form, li, ol, ul, p, pre, table) - начинаются с новой строки
  • Строчные: (span, b, a, strong, img, input, em) - пишутся без переноса
Div - контейнер - используется для задания стиля блокам контейнера.
Span - контейнер - используется для задания стиля отдельных частей текстка.
div и span - используется как контейнер для других элементов HTML. Div может содержать несколько элементов span. Span не может содержать div!!.
